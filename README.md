## 2. 概述
* 由于业务发展，预计公司明年的项目会有高性能NoSql数据库应用要求，我前两年在公司引入hbase+hive+impala还是稍显笨重，灵活性和性能都已经不满足业务需要。经过选型考察，我的目标瞄准了MongoDB。
